# Chrome Web Store Screenshots

Store your Chrome Web Store screenshots here.

## Requirements

- **Dimensions**: 1280x800 or 640x400 (recommended: 1280x800)
- **Format**: PNG or JPEG
- **Limit**: 5 screenshots max
- **File size**: Max 10MB each

## Naming Convention

Use descriptive names that indicate the order and content:

- `01-hero-main-interface.png` - Main interface showing the tree structure
- `02-drag-and-drop.png` - Demonstrating drag & drop functionality
- `03-nested-tabs.png` - Multi-level nesting example
- `04-search-filter.png` - Search functionality (if available)
- `05-context-menu.png` - Right-click context menu

## Tips

- Show real-world use cases (research, development, etc.)
- Use clear, readable text sizes
- Avoid cluttered examples
- Consider adding brief text overlays explaining features
- First screenshot is most important (it's the hero image)

## Updating

When screenshots change:
1. Add new screenshots to this directory
2. Update the Chrome Web Store listing manually
3. Note the update date in git commit
