# What's New

This file tracks the "What's new" section for Chrome Web Store updates.

## Template for Updates

When releasing a new version, copy the relevant section from CHANGELOG.md and format it for the store listing (max ~500 words).

## Current Version (Example)

### Version 0.1.2

Infrastructure and workflow improvements:

- Set up automated release management with Changesets
- Improved publishing workflow for faster updates
- Better GitHub Actions integration with detailed notifications
- Enhanced documentation for contributors

Full changelog: https://github.com/firtoz/tab-canopy/releases

## Guidelines

- Keep it concise and user-focused
- Highlight user-visible changes (not infrastructure)
- Link to GitHub releases for full details
- Update this file with each release before publishing
