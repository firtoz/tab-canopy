# Firefox Release Notes

This file tracks release notes for Firefox Add-ons submissions.

## Template for Updates

When releasing a new version, copy the relevant section from CHANGELOG.md and format it for AMO.

## Current Version (Example)

### Version 0.1.2

Infrastructure and workflow improvements:

- Set up automated release management with Changesets
- Improved publishing workflow for faster updates
- Better GitHub Actions integration
- Enhanced documentation for contributors

Full changelog: https://github.com/firtoz/tab-canopy/releases

---

## Guidelines

- Keep it concise and user-focused
- Highlight user-visible changes
- Link to GitHub releases for full details
- Update this file with each release

## Note

Firefox Add-ons API supports programmatic metadata updates, so we may automate this in the future.
