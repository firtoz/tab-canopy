# Firefox Add-ons Screenshots

Store your Firefox Add-ons screenshots here.

## Requirements

- **Dimensions**: Various supported, recommended 1280x800
- **Format**: PNG or JPEG
- **Limit**: Multiple screenshots allowed
- **File size**: Reasonable file sizes

## Naming Convention

Use descriptive names that indicate the order and content:

- `01-hero-main-interface.png` - Main interface showing the tree structure
- `02-drag-and-drop.png` - Demonstrating drag & drop functionality
- `03-nested-tabs.png` - Multi-level nesting example
- `04-search-filter.png` - Search functionality (if available)
- `05-context-menu.png` - Right-click context menu
