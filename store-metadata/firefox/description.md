# Tab Canopy - Firefox Add-ons Description

**Character limits: 250 for summary, unlimited for detailed**

## Summary (250 chars max)

Organize your browser tabs in a hierarchical tree structure with intuitive drag-and-drop. Create multi-level tab hierarchies, collapse branches, and never lose track of your work again. Privacy-first, open source.

## Detailed Description

Lost in a forest of browser tabs? Tab Canopy brings hierarchical organization to Firefox with an intuitive tree-based interface.

### Key Features

**Tree-Based Organization**
- Nest tabs under parent tabs to create logical hierarchies
- Multi-level nesting for complex project structures
- Collapse/expand branches to keep your workspace clean

**Drag & Drop Interface**
- Reorder tabs by dragging up or down
- Nest tabs by dragging onto parent tabs
- Un-nest by dragging back to the left
- Move entire branches at once

**Fast Navigation**
- Click any tab to instantly switch to it
- Search functionality to quickly find tabs
- Visual tree structure makes tabs easy to locate

**Privacy-First**
- All data stored locally in your browser
- No external servers or data collection
- Open source and transparent

### Perfect For

- Research workflows with multiple sources
- Development (documentation, code, tools)
- Content creation and writing projects
- Learning and course work
- Project management with related tabs

### How It Works

1. Click the Tab Canopy icon to open the side panel
2. See all your tabs in a tree structure
3. Drag tabs to reorganize or nest them
4. Click any tab to navigate instantly

### Open Source

Tab Canopy is open source and actively developed. Contributions welcome!

- GitHub: https://github.com/firtoz/tab-canopy
- Full changelog: https://github.com/firtoz/tab-canopy/releases
- Report issues: https://github.com/firtoz/tab-canopy/issues

### Permissions

- **tabs**: Required to read and manage browser tabs
- **sidePanel**: Required to display the side panel interface

No host permissions required - Tab Canopy does not read or modify web page content.
