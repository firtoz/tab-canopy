# Documentation Assets

This folder contains media assets for the README and documentation.

## Files

- `demo.gif` or `demo.mp4` - Main hero demo showing Tab Canopy in action
- `screenshots/` - Additional feature screenshots and images

## Git LFS

All media files in this directory are tracked by Git LFS to keep the repository size manageable. When you add new images or videos, they will automatically be stored in LFS.

To verify LFS is working after adding a file:
```bash
git lfs ls-files
```
